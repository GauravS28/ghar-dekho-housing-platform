# 🏠 Ghar Dekho - Housing Platform

A modern real estate web application that allows users to search, explore, and manage property listings with a clean and user-friendly interface.

---

## 🚀 Features

- 🔍 Search and explore property listings
- 🏘️ View detailed property information
- 👤 User authentication (Login/Register)
- 📝 Create and manage property listings
- 💬 Real-time communication (Socket integration)
- 📱 Responsive and user-friendly UI

---
